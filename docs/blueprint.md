# **App Name**: KisanGPT

## Core Features:

- Image Upload: Simple UI for uploading crop images.
- Disease Diagnosis: Uses the Gemini multimodal model to analyze uploaded images for plant diseases.
- Remedy Suggestion: The tool provides simple and affordable remedies suitable for small farmers in India, generated by Gemini.
- Results Display: Displays the diagnosis and suggested remedies in a clear, easy-to-understand format.
- Text-to-Speech: Optionally, allow users to convert the text response into voice output (available in multiple Indian languages).
- Market Price Trend Summarization: Integrate with Agmarknet to summarize market trends and give price predictions
- Government Scheme Information: A tool that Uses Gemini to provide eligibility, application process, and status updates for PM Kisan and other state-specific schemes.

## Style Guidelines:

- Primary color: Earthy green (#388E3C) to evoke nature and growth.
- Background color: Light beige (#F5F5DC), a desaturated, bright tint of the primary green.
- Accent color: Warm gold (#D4AC0D) for important highlights, contrasting against the earthy green, but harmonious with it.
- Body text and headline Font: 'PT Sans' (sans-serif) which is modern and readable.
- Code Font: 'Source Code Pro' for displaying code snippets.
- Use simple, clear icons to represent different functions.
- Clean and intuitive layout with clear sections for image upload, results display, and additional information.

# KisanGPT

KisanGPT is an AI-powered farming assistant designed to help Indian farmers with disease diagnosis, market trends, government schemes, weather forecasts, and more.

## Features

- **AI Disease Diagnosis**: Upload photos of plants to identify diseases and get remedies.
- **Market Trends**: Get real-time summaries and future price predictions for crops.
- **Government Schemes**: Personalized recommendations for relevant agricultural schemes.
- **Weather Forecast**: Accurate 3-day local weather forecasts.
- **Soil Testing**: Analyze soil health via description or imagery.
- **Farmer Community**: A forum for farmers to connect and share knowledge.
- **Multi-language Support**: Available in English, Hindi, and Marathi.

## Tech Stack

- **Framework**: Next.js (App Router)
- **UI Components**: ShadCN UI, Tailwind CSS
- **AI**: Genkit with Google Gemini models
- **Backend**: Firebase (Authentication & Firestore)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/BHAVESH0704/KisanGPT.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables in a `.env` file (Firebase and Gemini API keys).
4. Run the development server:
   ```bash
   npm run dev
   ```

Explore the app at `http://localhost:3000`.
